`Colorspace` describes a pixel colorspace. The DeviceGray / DeviceRGB /
DeviceCMYK singletons are re-exported as
[`csGRAY` / `csRGB` / `csCMYK`](/docs/reference/constants#colorspace-singletons--font-tables).
